<h1>Pantry Pal</h1>
<p>Pantry Pal is an app designed to help you keep track of the items in your pantry and help you keep track of your grocery shopping needs.<br><br>Some of the features include:
<ul>
  <li>Expiry Date Warnings</li>
  <li>Recipe Tracking</li>
  <li>Pantry Tracking</li>
  <li>Safe and remote data storage using firebase</li>
</ul>

This project was built as part of the Advanced Topics in Mobile Development course using tools such as React Native, Expo, and Firebase.
</p>
